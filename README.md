# tweepy-bots
README.md

Prerequisites
1. Twitter Account
2. Tweepy, PIL

Create a Twitter App
1. Go to apps.twitter.com and click on "Create New App" button
2. On "Keys and Access Tokens" tab click on "Create my access token"
3. Save the cunsomer key, key secret, access token and access token secret

These bots we will use Tweepy, install with
# pip install tweepy
